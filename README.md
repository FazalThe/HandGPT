# HandGPT
A handheld device to talk to AI with ESP32 S3 WROOM(N16R8) DEVKITC.
Well, why not just use your phone? This is a distraction free way to talk to AI.Who knows, if you got carried away and locked away your phone somewhere to lock in but you are living in 2026.

#BOM
- ESP32 S3 WROOM(N16R8) DEVKITC
- TP4056
- ILI9341 TFT LCD 2.4 inch
- INMP441 Mic module
- 12x12 Momentary tactile 4-pin Pushbutton (x2)
- 504030 (size is changable) 3.7 V li-po Battery
